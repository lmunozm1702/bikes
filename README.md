## 💻 Getting Started <a name="getting-started"></a>

> In this app, we call the providers API to get the list of companies aroun the worls. Once we have the data into the redux store, we group them by conutry and filter by continet, using Europe as main and default filter. Each country info can be clicked and then the app displays the companies by city on the selected country.

- Clone the project from this link: [https://github.com/lmunozm1702/bikes](https://github.com/lmunozm1702/bikes)

### Prerequisites

In order to run this project you need:

- Web browser
- Code editor
- Good internet connection

### Setup

Clone this repository to your desired folder by using the following command:

```sh
  cd my-folder
  git clone https://github.com/lmunozm1702/bikes.git
```

### Install

Install these npm dependencies by running the commands below:

```sh
  cd bikes
  npm install
```

### Usage

To run the project, execute the following command:

```sh
  npm start
```

This will open the browser and be able to pull data from the server

### Run tests

To run tests, run the following command:

```sh
  npm test
```
